# TO-DO-LIST
# 🚀 Git Task Log – GitHub Inspired To-Do List

A modern **GitHub-inspired To-Do List** built using **HTML, CSS, and JavaScript**. This project transforms everyday task management into a developer-friendly experience by using Git concepts such as **Stage**, **Commit**, **Commit Hash**, and **Revert**.

## ✨ Features

- 📝 Add new tasks
- ✅ Mark tasks as completed
- 🗑️ Delete tasks (Revert)
- 💾 Automatically saves tasks using Local Storage
- 🔖 Random Git-style commit hash for completed tasks
- ⌨️ Press **Enter** to quickly add tasks
- 📱 Fully Responsive Design
- 🎨 Clean GitHub-inspired User Interface
- ⚡ Lightweight and Fast

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/DineshRaj74/TO-DO-LIST.git
```

2. Open the project folder

```bash
cd TO-DO-LIST
```

3. Open `index.html` in your browser.
